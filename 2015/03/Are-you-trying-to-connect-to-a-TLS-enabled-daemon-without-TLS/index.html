Docker

FATA[0000]Are you trying to connect to a TLS-enabled daemon without TLS


从 Vagrant 的深坑掉入了 Docker 的深坑，OS X 使用 Boot2Docker 来使用 Docker，直接从官网下载后安装出现了上面的错误，但是Google之发现几乎没有人遇上过…

多次重启 Boot2Docker 无解，运行 Installation Guide 里的

$ docker run hello-world


也是同样地错误，最后重启 VirtualBox 并且

sudo boot2docker -v up


问题似乎得到了解决。
update 2015-03-01 17:17:12
问题再次发生，运行

$(boot2docker shellinit)


似乎可以解决这个问题。
如果问题持续出现，可以尝试将 ‘$(boot2docker shellinit)’ 加入 ‘~/.profile’ 中
